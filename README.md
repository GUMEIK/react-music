这个项目是利用react,redux,antd-mobile,以及一些中间件如react-redux,redux-logger,redux-thunk等
# 安装和使用
```git clone https://github.com/GUMEIK/react-music.git```
```npm install```
```npm start```
后端服务器采用的是```NeteaseCloudMusicApi```
